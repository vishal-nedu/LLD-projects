# LLD-projects
1. Snake Game

Folder: snake-game

Description

A console-based Snake Game implementation. Demonstrates:

Object-oriented design with classes like Snake, Food, and GameBoard

Game loop management

Collision detection and score tracking

Features

Move snake in four directions

Eat food to grow

Detect collisions with walls or self

Display score in console

2. Taxi Booking System

Folder: taxi-booking

Description

A console-based Taxi Booking System. Focuses on managing taxis, drivers, and rides using OOP concepts.

Features

Book a taxi for a ride

View available taxis and drivers

Assign drivers automatically based on availability

Track ride status: Booked, In Progress, Completed

3. Library Management System

Folder: library-management

Description

A simple Library Management System for managing books, borrowing, and returning. Demonstrates class relationships and menu-driven workflows.

Features

Add, view, borrow, and return books

Track available copies

Menu-driven console interface

Simple, easy-to-understand low-level design

4. Railway Ticket Reservation System

Folder: ticket-reservation

Description

A console-based Railway Ticket Reservation System. Demonstrates advanced LLD concepts like:

Booking with preferences (Lower, Middle, Upper berths)

RAC (Reservation Against Cancellation) handling

Waiting list management

Child ticket handling linked to adult tickets

Automatic upgrading from RAC or Waiting List to confirmed tickets

Features

Book, cancel, and view tickets

View available berths

Handle RAC and waiting lists

Print booked passenger details
